# todo.c
A lightweight gtk based todo application

## Installation

```fish
git clone https://github.com/quantumvoid0/todo.c
cd todo.c
make install
```

## Uninstallation

```fish
git clone https://github.com/quantumvoid0/todo.c
cd todo.c
make uninstall
```

## Usage

just run `todo` in ur terminal or bind a key to it.
